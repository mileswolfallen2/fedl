502 Status

Bad Gateway

A server acting as a gateway or proxy received an invalid response from somewhere upstream.

This is a good fit for integrations that depend on external APIs or cached edge services.

Go Home More Status Pages

Likely Causes

  • Upstream API failedAn external data source may have returned malformed data.
  • Proxy issueAn edge layer may be misconfigured or temporarily unstable.

Tip: 502 usually points to infrastructure between your app and another service, not the browser itself.
