Common HTTP Error Pages

Preview the shared error page designs below. These pages are set up as standalone static HTML files so you can use them directly or wire them into hosting rules later.

401 Unauthorized. Sign-in or permission issue. 403 Forbidden. Access is blocked even if the page exists. 404 Not Found. The requested page could not be located. 429 Too Many Requests. The user should slow down and retry later. 500 Internal Server Error. Something failed on the server side. 502 Bad Gateway. An upstream response was invalid. 503 Service Unavailable. The app is down or under maintenance.
